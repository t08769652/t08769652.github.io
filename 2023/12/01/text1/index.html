钓鱼网站链接获取

发表于2023-12-01更新于2023-12-03

1.vim /etc/proxychains4.conf (修改socks 4-5 更改IP和端口号)
2.proxychains4 git clone https://github.com/makdosx/mip22 (下载需要的文件)
3.cd mip22
4.chmod +x ./* (增加执行权限)
5.proxychains4 bash ./mip22.sh (执行脚本文件)
出现底下情况执行 proxychains4 -q bash ./mip22.sh

6.输入1
7.输入数字选择需要的网站

8.根据数字选择链接在内网还是外网生效
